## Installation

<Tabs
defaultValue="apple-intel"
values={[

<TabItem value="apple-intel">

Download latest release.
```shell
curl -LO https://release.ariga.io/atlas/atlas-darwin-amd64-v0.0.1
```

Make the atlas binary executable.
```shell
chmod +x ./atlas-darwin-amd64-v0.0.1
```

Move the atlas binary to a file location on your system PATH.
```shell
sudo mv ./atlas-darwin-amd64-v0.0.1 /usr/local/bin/atlas
```
```shell
sudo chown root: /usr/local/bin/atlas
```

</TabItem>

<TabItem value="linux">

Download latest release.
```shell
curl -LO https://release.ariga.io/atlas/atlas-linux-amd64-v0.0.1
```

Move the atlas binary to a file location on your system PATH.
```shell
sudo install -o root -g root -m 0755 ./atlas-linux-amd64-v0.0.1 /usr/local/bin/atlas
```

</TabItem>
