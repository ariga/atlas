## `dictionary` {#dictionary}
The `dictionary` block describes a dictionary in a database schema.

~~~hcl
dictionary "name" {
  schema = schema.public
  source = sql("CLICKHOUSE(TABLE)")
  layout = sql("HASH()")
  key "id" {
	type = UInt64
  }
  attribute "name" {
	type = String
  }
  lifetime {
	min = 0
	max = 0
  }
  ...
}
~~~
