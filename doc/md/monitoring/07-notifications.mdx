---
title: Notifications
---

To receive notifications about schema changes, you can set up notifications in Atlas Schema Monitoring. Notifications can be sent to Slack, Workplace, or a plain webhook.


#### Setting up Notifications

To set up notifications, go to the **webhooks** tab of the database you want to monitor and click on the **Add Webhook** button.

Choose the type of notification you want to receive, and provide the necessary information to set up the notification.
