title: Quickstart
---

A quick and easy to follow guide to get started.

1. Spin up a local docker with Postgres and a large schema
2. Install the Atlas agent locally
3. Connect the agent to the Postgres database
4. View the schema in the Atlas UI
5. Make a change to the schema and see it reflected in the Atlas UI
